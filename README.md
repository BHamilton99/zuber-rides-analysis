# Zuber Ride-Sharing Data Analysis Project

## Overview
This project analyzes ride-sharing data to understand market demand and evaluate how weather conditions impact ride duration.

## Objective
- Identify top taxi companies by ride volume
- Analyze high-demand neighborhoods
- Evaluate the effect of weather on ride duration
- Test statistical significance of observed differences

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- SQL
- Statistical Testing (t-test)

## Methodology
- Performed exploratory data analysis on taxi and neighborhood datasets
- Visualized ride distribution across companies and locations
- Filtered and analyzed rides between Loop and O’Hare
- Conducted hypothesis testing using Welch’s t-test

## Key Results
- Identified top-performing taxi companies and high-traffic neighborhoods
- Found a significant increase (~21%) in ride duration during rainy conditions

## Conclusion
Weather conditions significantly impact ride duration, providing insights for pricing and operational strategies.
